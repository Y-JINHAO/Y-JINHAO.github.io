3 API

三种方式使 AOS 对象成为全局变量：

  • init - 初始化
  • refresh - 重新计算元素的所有偏移量和位置（在窗口调整大小时调用）
  • refreshHard - 使用 AOS 元素和触发器重新初始化数组 refresh（调用与 DOM 变化相关的 aos 元素）

4 JS Events

元素在进行动画处理时，AOS 可以在文档上调度两个事件：aos:in和aos:out 用来实现一些额外的功能。
